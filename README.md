# O que é ?

O Lightbot é um bot do modificado do [MhankBar](https://github.com/MhankBarBar/termux-wabot) e traduzido para o português pt-br, e usado no termux e é bem facil a instalação, segue os comandos de como instalar o Lightbot

---


# Instalação

[termux](https://play.google.com/store/apps/details?id=com.termux&hl=pt_BR&gl=US)

```
git clone https://github.com/devjavadk/githubbot
cd githubbot
bash install.sh
npm start
```
